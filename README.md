This chrome extension replaces (Bug|bug|バグ|ばぐ) to (Dragon|dragon|ドラゴン|どらごん).

Inspired by [バグをドラゴンと呼ぶ運用を始めて1ヶ月くらいたった](http://konifar.hatenablog.com/entry/2015/05/02/003449).
